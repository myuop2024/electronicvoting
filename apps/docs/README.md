# ObserverNet Election Platform Documentation

This documentation site outlines architecture decisions, operational playbooks, and developer onboarding steps for the ObserverNet multi-tenant election platform.

## Contents

- `architecture.md` – System overview, data flows, security posture
- `runbooks/` – Operational runbooks for blockchain, verification, and messaging providers
- `playbooks/` – Incident response and anomaly investigation guides
- `load-testing.md` – k6 profiles and tuning recommendations for one million concurrent voters

Docs are authored in Markdown and can be rendered using any static site generator (e.g., Nextra) when promoted to production.
